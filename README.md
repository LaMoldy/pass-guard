# pass-guard
This project is a simple password manager that is written in Python called PassGuard. This will allow you to generate and store passwords using a custom made interface that you can use for any account.

## Table of Contents:
- [Installation](#installation)
- [Usage](#usage)
- [Contributing](#Contributing)
- [License](#license)

## Installation:
Clone the repository in a location that you want the project stored:
```bash
git clone https://github.com/LaMoldy/pass-guard.git
```

## Usage
To use this application:
1. Open a command prompt or terminal
2. Change to the directory PassGuard is located
3. Run the following command:
```bash
python main.py
```
